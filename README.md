# Handwritten Digit Recognition with MNIST

This project involves training a neural network to recognize handwritten digits using the MNIST dataset. The model is built using TensorFlow and Keras, and it is capable of predicting digits from custom images.

## Requirements

To run this project, you will need the following libraries:

- TensorFlow
- NumPy
- OpenCV
- Matplotlib

You can install these dependencies using pip:

```bash
pip install tensorflow numpy opencv-python matplotlib
